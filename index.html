How to Use

Note:

You firstly need to enable the Developer Settings of your Android device.

Wired Connection

  1. 1
    Enable the USB Debugging in the Developer Settings page
  2. 2
    Connect your device with computer via a USB cable
  3. 3
    Just run the executable and skip the pairing and connecting steps
  4. 4
    Enjoy your mouse and keyboard on Android device!

Wireless connection

  1. 1
    Enable the Wireless Debugging in the Developer Settings page
  2. 2
    Run the executable
  3. 3
    Input the IP address, port, and pairing code into the pairing tab of the connecting window
  4. 4
    Input the IP address and port from the main Wireless Debugging into the connecting tab
  5. 5
    Enjoy your mouse and keyboard on Android device!

Shortcuts

Shortcuts below are available while sharing:

Shortcut Description
Ctrl+Alt+S Toggle the sharing state
Ctrl+Alt+Q Quit the program
F1 Multi-task switching
F2 Return to Home
F3 Back
F4 Previous Media
F5 Play / Pause Media
F6 Next Media
F7 Volume Down
F8 Volume Up
F9 Brightness Down
F10 Brightness Up
F11 Screen Sleep
F12 Wake Up

Shortcuts below are available while not sharing:

Shortcut Description
Alt+↑ Scroll Up
Alt+↓ Scroll Down
Alt+[ Previous Media
Alt+] Next Media
Alt+\ Play / Pause Media
